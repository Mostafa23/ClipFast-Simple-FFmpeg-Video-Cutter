ClipFast: Simple FFmpeg Video Cutter
============================================================

A lightweight, fast, and user-friendly desktop app built with Python (PySide6) for lossless video cutting powered by FFmpeg.
Designed for simplicity, speed, and precision.

KEY FEATURES
------------------------------------------------------------
- Ultra-Fast & Lossless: Uses 'ffmpeg -c copy' to cut without re-encoding.
- Clean Interface: Minimal, intuitive GUI built with PySide6.
- Accurate Timing: Define cut points using the 'HH:MM:SS' format.
- Drag & Drop Support: Quickly load video files.
- Cross-Platform: Runs on Windows, macOS, and Linux.

REQUIREMENTS
------------------------------------------------------------
1. FFmpeg Installed and in PATH:
   - Windows: Download binaries and add to System PATH.
   - Linux (Ubuntu/Debian): sudo apt install ffmpeg
   - macOS (Homebrew): brew install ffmpeg

2. Python Packages:
   pip install PySide6

GETTING STARTED
------------------------------------------------------------
1. Clone the repository:
   git clone https://github.com/Mostafa23/ClipFast-Simple-FFmpeg-Video-Cutter.git

3. Run the app:
   python ClipFast.py

HOW TO USE
------------------------------------------------------------
1. Open Video: Drag and drop your file or click to browse.
2. Set Time Range: Enter start and end times (HH:MM:SS).
3. Cut & Save: Click 'Cut Video' and choose output name and location.
